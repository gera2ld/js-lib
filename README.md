# js-lib

JavaScript utilities for use in browser and Deno.

## Usage

```html
<script type="importmap">
  {
    "imports": {
      "js-lib/": "https://cdn.jsdelivr.net/gh/gera2ld/js-lib@dist/"
    }
  }
</script>

<script type="module">
import 'js-lib/unocss';
import 'js-lib/web-font';
</script>
```
